# Legacy Responsive Web Design (V8)

This repository contains responsive web design projects completed as part of **freeCodeCamp’s Legacy Responsive Web Design (V8) curriculum**. All projects are implemented using **HTML** and **CSS**, with an emphasis on clean structure, responsive layouts, and accessibility best practices.

## Projects

The repository includes the following certification projects:

- **Survey Form** – Building accessible forms using semantic HTML and CSS  
- **Tribute Page** – Structuring content with semantic HTML and basic layout techniques  
- **Technical Documentation Page** – Creating structured documentation with navigation  
- **Product Landing Page** – Designing a responsive landing page with embedded media  
- **Personal Portfolio Webpage** – A responsive portfolio showcasing completed projects  

Each project is self-contained within its own directory.

## Technologies Used

- HTML5  
- CSS3  

No external frameworks or libraries are used.

## Purpose

This repository serves as:
- Coursework for the freeCodeCamp Responsive Web Design certification  
- A reference for foundational front-end web development concepts  
- A portfolio of responsive, standards-compliant webpages
